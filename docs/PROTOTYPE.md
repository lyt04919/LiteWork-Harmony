# LiteWork 团队协作管理 App —— 界面原型与设计规范

本文件整理了 LiteWork 应用的界面设计原型、组件布局规范以及关键交互设计说明。

原型截图保存在本地目录：[docs/prototypes/](file:///Users/liyuteng/Documents/finalProject/docs/prototypes/)

---

## 1. 首页任务看板 (Task Board)
* **原型文件**：`docs/prototypes/task_board.png`
* **布局规范**：
  * **顶部搜索栏**：双维度模糊搜索输入框，支持以“任务标题”、“负责人姓名”为关键字实时过滤。
  * **状态切换卡 (Tabs)**：提供“待办”、“进行中”、“已完成”三个标签页。通过横向滑动或点击快速过滤对应状态的卡片。
  * **负责人筛选**：Tabs 下方提供负责人快速筛选横向滚动条，可单选某位团队成员或选择“全部”。
  * **任务列表**：
    * 列表为空时展示插画与缺省提示词。
    * 卡片具备 12px 圆角、1px 细边框，以及浅层阴影以保证卡片层次感。
    * 卡片包含任务标题（粗体）、截止时间（带图标）、负责人头像/姓名徽章、优先级标签（高优先级用淡红，普通用淡蓝）。
  * **新建任务悬浮按钮 (FAB)**：位于屏幕右下角，使用高亮的 Tech Blue (#0066FF) 圆形按钮，带有 `+` 图标，点击后平滑跳转至任务创建页。
* **交互细节**：
  * 长按卡片可触发系统拖动反馈，卡片浮起半透明，上下拖动可自由调整任务在当前状态下的排序。

## 2. 新建/修改任务表单 (Create Task Form)
* **原型文件**：`docs/prototypes/create_task.png`
* **布局规范**：
  * **顶部导航栏**：包含返回按钮和页面标题（新建任务 / 编辑任务）。
  * **表单内容区**：
    * **任务标题输入框**：支持占位提示符，当为空且点击保存时，会触发 Toast 弱提示拦截。
    * **描述输入域**：多行文本域，支持详细的工作备注。
    * **负责人选择器**：下拉菜单组件（`Select`），内置团队成员名单。
    * **优先级选择器**：下拉菜单组件，可选“普通”或“高”。
    * **状态选择器**：下拉菜单组件（仅在编辑模式下可见，以便调整状态）。
    * **截止日期选择器**：横向 Row 行，点击后拉起 HarmonyOS 原生 `DatePickerDialog` 底部日期选择轮盘。
  * **操作按钮组**：表单底部提供“取消”（次级边框按钮）与“保存任务”（主色填充按钮）。

## 3. AI 智能客服 (LiteAI Chat)
* **原型文件**：`docs/prototypes/ai_chat.png`
* **布局规范**：
  * **顶部标题栏**：显示“AI 助手 LiteAI”以及“在线”状态。
  * **消息历史滚动区**：
    * **AI 气泡**：左侧对齐，灰色背景（亮色模式）或深灰色背景（暗色模式），文字左对齐。
    * **用户气泡**：右侧对齐，Tech Blue 背景，白色文字，文字右对齐。
    * **状态指示**：当 AI 在进行接口调用时，底部展示“AI 正在输入...”并显示闪烁打字态。
  * **底部输入栏**：包含左侧小图标、文本输入框以及右侧圆形的“发送”按钮。支持点击软键盘回车键一键发送。

## 4. 个人中心 (Profile)
* **原型文件**：`docs/prototypes/profile.png`
* **布局规范**：
  * **头部用户信息卡**：包含用户圆形头像、用户昵称及个人简介。
  * **本周完成度看板**：
    * 采用圆形环图（HarmonyOS `DataPanel` 组件），蓝色弧线部分表示已完成任务的比例，灰色代表未完成。
    * 环图中心动态展示数值（例如：`80%`，并在下方标明已完成任务数和总任务数）。
  * **设置导航列表**：
    * 包含“系统设置”、“关于我们”等导航栏目。
    * 点击“系统设置”可跳转至主题切换页。
